
# Golang Kafka Shovel Project

Shoveler project for Kafka

## Installation


```bash
  git clone https://github.com/burakolgun/gokafkashovel.git
```


## Run Locally

Go to the project directory

```bash
  cd gokafkashovel
```

```bash
  docker-compose -up -d
```

Install dependencies

```bash
  go mod download
```

Start the server

```bash
  go run
```
